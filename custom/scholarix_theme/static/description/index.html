Quick Installation

Get started with Scholarix AI Theme in just a few simple steps:

  1. Copy the scholarix_theme folder to your Odoo addons directory
  2. Update your Odoo configuration to include the addons path
  3. Install the theme through Odoo Apps menu
  4. Activate the theme in Website Settings
  5. Customize colors, fonts, and components as needed

Detailed installation instructions and customization guide are included in the README.md file.
